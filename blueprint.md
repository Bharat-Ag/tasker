# **App Name**: Daily Plan Generator

## Core Features:

- Project Input: Allow users to input project names.
- Task Input: Allow users to input tasks associated with each project.
- Plan Generation: Generate a formatted work plan including a greeting and project/task list.

## Style Guidelines:

- Primary color: Deep navy blue (#2E3A87), reflecting professionalism and focus.
- Background color: Soft off-white (#F8F9FA), providing a clean and non-distracting backdrop.
- Accent color: Muted teal (#34A7A7), for subtle visual interest and important actions.
- Body and headline font: 'PT Sans', a humanist sans-serif combining modernity and a touch of warmth for readability and approachability.
- Clean, well-structured layout to promote readability.
